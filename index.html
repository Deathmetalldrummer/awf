awf by dobedo

A widget factory

'A widget factory' is a theme preview application for gtk2 and gtk3. It displays the various widget types provided by gtk2/gtk3 in a single window allowing to see the final visual effect of the applied theme.

Dependencies

gtk2 and/or gtk3

Install

./configure (to build the gtk2 version) ./configure --with-gtk3 (to build the gtk3 version) make make install
